# 2557번 Hello World
## 문제 
Hello World! 출력   
https://www.acmicpc.net/problem/2557

## 알아야 할 내용
1. 출력 함수 
    - 파이썬에서 `print()` 함수는 덱스트나 값을 화면에 출력할 때 사용됩니다.

    - 이 문제에서는 "Hello World!"라는 **문자열**을 출력합니다.

2. 문자열 
    - 문자열은 **큰따옴표**나 **작은따옴표**로 감싸서 표현합니다    

## 코드
```python
print("Hello World!")
```
